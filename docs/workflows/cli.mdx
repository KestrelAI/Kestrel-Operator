The Kestrel CLI lets you manage workflows from your terminal. Use it for scripting, CI/CD integration, and day-to-day workflow management.

## Installation

```bash
curl -fsSL https://get.usekestrel.ai/cli | sh
```

Or install via Homebrew:

```bash
brew install kestrelai/tap/kestrel
```

## Authentication

Authenticate with your Kestrel email and password:

```bash
kestrel login
```

You'll be prompted for your server URL, email, and password. Credentials are stored locally at `~/.kestrel/config.json`.

```bash
kestrel login --server https://app.usekestrel.ai --email you@company.com
```

Check your current session:

```bash
kestrel status
```

Log out and clear stored credentials:

```bash
kestrel logout
```

## Workflow Commands

### List workflows

```bash
kestrel workflows list
kestrel workflows list --status active
```

Aliases: `kestrel wf list`

### Get workflow details

Shows the workflow definition, trigger config, and an ASCII diagram of the workflow DAG:

```bash
kestrel workflows get <workflow-id>
```

### Create a workflow from file

Create a workflow from a JSON definition:

```bash
kestrel workflows create --file workflow.json
```

### Generate from natural language

Use the AI agent to generate a workflow from a description:

```bash
kestrel workflows generate "When a pod crashloops, run RCA, create a Jira ticket, and notify #incidents on Slack"
```

Add `--save` to immediately save the generated workflow as a draft:

```bash
kestrel workflows generate --save "When a PagerDuty alert fires, investigate the Kubernetes cluster and post findings to Slack"
```

### Edit a workflow

Update a workflow's name or description:

```bash
kestrel workflows edit <workflow-id> --name "New Name"
kestrel workflows edit <workflow-id> --description "When a pod crashloops, run RCA and notify Slack"
```

### Delete a workflow

```bash
kestrel workflows delete <workflow-id>
```

### Activate / Pause

```bash
kestrel workflows activate <workflow-id>
kestrel workflows pause <workflow-id>
```

### Duplicate a workflow

```bash
kestrel workflows duplicate <workflow-id>
kestrel workflows duplicate <workflow-id> --name "Copy of my workflow"
```

### Test a workflow

Run a workflow with mock trigger data:

```bash
kestrel workflows test <workflow-id>
```

### Request a workflow

Submit a natural language request to trigger a matching workflow. This is the CLI equivalent of the Slack `/kestrel-workflow` command:

```bash
kestrel workflows request "provision an MSK cluster with 3 brokers in us-east-1"
kestrel workflows request "scale the payments deployment to 5 replicas"
```

If a matching workflow is found, it executes immediately. If parameters are missing, you'll be prompted interactively. If no match exists, the request is logged for your infrastructure team.

## Execution Commands

### List executions for a workflow

```bash
kestrel workflows executions <workflow-id>
kestrel workflows executions <workflow-id> --page 2 --page-size 10
```

### View workflow statistics

```bash
kestrel workflows stats
```

Shows total/active workflows, execution counts, status breakdown, and a daily execution trend.

## Approval Commands

### List pending approvals

```bash
kestrel approvals list
```

### Approve a workflow step

```bash
kestrel approvals approve <approval-id>
```

For justification-type approvals, provide a reason:

```bash
kestrel approvals approve <approval-id> --justification "Emergency fix for production outage"
```

If you omit `--justification` on a justification-type approval, you'll be prompted interactively.

### Reject a workflow step

```bash
kestrel approvals reject <approval-id>
```

## Request Commands

### List unmatched requests

View workflow requests that couldn't be matched to an existing workflow:

```bash
kestrel requests list
```

### Approve / Reject requests

```bash
kestrel requests approve <request-id>
kestrel requests reject <request-id>
```

## Catalog and Integrations

### View available triggers and actions

```bash
kestrel workflows catalog
```

### Check integration status

```bash
kestrel workflows integrations
```

### View AI-suggested workflows

```bash
kestrel workflows suggestions
```
